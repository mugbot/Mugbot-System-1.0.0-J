www.mugbot.com
の「マグボットの作り方」に掲載しているソーシャルロボットマグボットの３つのソフトです。
I updated the original Mugbot README file by adding English notation. I used Google Translate for all translations.
mugbot_web_v1.0.0_.html
PC、タブレット、スマホのブラウザーから操作するマグボットの操作インタフェースです。
The mugbot_web_v1_0_0_ENG.html now has English notation on the buttons and fields. It may need correcting. This web page should allow a user to update Mugbot from any web browser on a computer or mobile device.

mugbot_server_v1.0.0_.rb
Raspberry Piの中に設置するRubyによるWebSocketサーバーです。
The mugbot_server_v1_0_0_ENG.rb is a server implementation written in Ruby that needs to be placed on the Raspberry Pi inside of Mugbot. This will receive the user input from the user input web page.

mugbot_arduino_v1.0.0.ino
Arduinoのプログラムです。
This is the Arduino file to be placed on the Mugbot Arduino.
